## 📁 Estructura del Proyecto

```
equipo-proyecto/
│
├── index.html              ← Página principal
│
├── css/
│   └── styles.css          ← Estilos personalizados (animaciones, etc.)
│
├── js/
│   └── main.js             ← Lógica del modal y datos del equipo
│
└── assets/
    └── img/
        ├── saul.jpg        ← Foto de Saul Adain Huillca Rodriguez
        ├── eduardo.jpg     ← Foto de Eduardo Luna Lezano
        ├── gari.jpg        ← Foto de Gari Joel Chavez Huanca
        └── cuarto.jpg      ← Foto del cuarto integrante (pendiente)
```

---

## 🖼️ Cómo agregar las fotos

1. Coloca las fotos de cada integrante dentro de la carpeta `assets/img/`.
2. Los nombres de los archivos deben coincidir exactamente:
   - `saul.jpg`
   - `eduardo.jpg`
   - `gari.jpg`
   - `cuarto.jpg` *(actualizar cuando se defina el cuarto integrante)*
3. Si no hay foto disponible, el sitio mostrará automáticamente un avatar de reemplazo.

> **Recomendación:** Usa imágenes cuadradas (ej. 400×400 px) para que se vean bien en el círculo de perfil.

## 🚀 Cómo subir a GitHub

1. Crea un repositorio en GitHub (puede ser público).
2. Sube **toda la carpeta `equipo-proyecto/`** manteniendo la estructura de carpetas.
3. Para ver el sitio en línea, activa **GitHub Pages** en la configuración del repositorio:
   - Ve a `Settings` → `Pages` → selecciona la rama `main` y la carpeta `/ (root)`.
   - GitHub te dará un enlace público para compartir.
